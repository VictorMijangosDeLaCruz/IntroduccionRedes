Datos estructurados¶

Los datos estructurados son aquellos datos que estn bien organizados bajo criterios especficos, a los cuales es fcil de acceder, as como son sencillos de procesor. Algunos tipos de datos estructurados tpicos son:

  • Bases de datos
  • Hojas de clculo (.csv, Esxcel)
  • Formularios

El manejo de datos estructurados es sencillo pues su misma natrualeza permite que se pueda acceder a ellos de manera simple. En python, una librera de gran utilidad para manejar datos estructurados es Pandas.

Datos desde hoja de cálculo¶

Pandas permite exportar datos desde una hoja de clculo en formato csv como:

pd.read_csv(file_name)

De forma similar, tambin permite leer datos en formato de hoja de clculo de Excel:

pd.read_excel(file_name)

Pandas tambin puede manejar datos que vengan de una base de datos SQL. Se pueden generar bases de datos por medio de to_sql tambin permite hacer consultas de la base de datos con read_sql.

from sqlite3 import connect

#Crea una base de datos
conn = connect(':memory:')
dataset.to_sql('dataset_db', conn)  
#Realiza consulta
pd.read_sql('SELECT Y FROM dataset_db', conn)

Reducción de dimensión para visualización¶

Dentro de la exploracin de datos, es muy comn que los datos de grandes dimensiones se reduzcan a dimensiones menores para poder visualizarse y poder explorar y explicar ciertos comportamientos en los datos. Para esto, se utilizan diferentes estrategias, entre una de las ms populares se encuentra el mtodo de PCA (Principal Component Analysis), el cul bsicamente se enfoca en encontrar una transformacin lineal $U^*$ que sea invertible y ortogonal. Es decir, se busca una funcin que cumpla:

$$U^* = \arg\min_U \frac{1}{2}|| x - U^T U x||$$

De tal forma que se satisfaga $U^T U \approx Id$$. Realizar esto lleva a un problema de valores y vectores propios. Para simplificar, usamos aqu la librera sklearn que ya cuenta con una implementacin de PCA.

El mtodo de PCA es equivalente a una red neuronal con varias restricciones. Ponemos a continuacin un cdigo que determina una funcin lineal (definida por una matriz $U$) que es equivalente al procesamiento que hace PCA, pero con un entrenamiento de red neuronal. Los resultados, si bien no los mismos, son similares.

In [13]:
import torch
import torch.nn as nn

#Datos a formato torch
X = torch.tensor(X, dtype=torch.float)

#Primera capa equivalente a U
red = nn.Linear(30, 2, bias=False)
#Segunda capa equivalente a U^T
decode = nn.Linear(2, 30, bias=False)

#Entrenamiento
risk = nn.MSELoss()
optimizer = torch.optim.Adam(list(red.parameters()) + list(decode.parameters()), lr=0.1)
for t in range(0,1000):
    reconst =  decode(red(X))
    optimizer.zero_grad()
    loss = risk(reconst, X)
    loss.backward()
    optimizer.step()
    

#Visualizacin
visualization = red(X).detach().numpy()
plt.scatter(visualization[:,0], visualization[:,1], c=Y)
plt.show()
